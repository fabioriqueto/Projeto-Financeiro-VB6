# Sistema Brusk Financeiro
## Sistema para efetuar transações financeiras, tais como trocas de cheques e empréstimos.
Linguagem utilizada: VB6 e banco de dados Access
Desenvolvido em novembro de 2000, este é um sistema financeiro para controle e calculos de trocas de cheques e empréstimos com arquivamento e impressões para controles bancários.
